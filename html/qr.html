This API can generate QR Code and you can customize width, height, dark color and light color of code image. API response to you with buffer image, so you can replace QR image link with this api.

API

https://maoapi.vercel.app/qrcode?width=200&height=200&dark=000000&light=ffffff&text=hello

Response image from above api

You can change query data of width, height and colors of QR Code. Both default width and height of QR Code is 200. Default dark color is #000000 and light is #ffffff
